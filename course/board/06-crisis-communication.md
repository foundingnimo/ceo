## Part 1: The Anatomy of a Crisis

Crises that affect board/investor relationships typically fall into these categories:

| Crisis Type | Examples | Severity |
| :--- | :--- | :--- |
| **Financial** | Cash runway shortfall, missed projections, fraud | High |
| **People** | Key executive departure, founder conflict, HR scandal | Medium-High |
| **Product** | Major failure, security breach, customer data loss | High |
| **Market** | Competitor disruption, market collapse, regulatory change | Medium |
| **Reputation** | PR disaster, legal action, public criticism | Medium-High |

### What Makes Crises Worse

| Mistake | Result |
| :--- | :--- |
| Hiding or downplaying | Trust destroyed when truth emerges |
| Slow communication | Investors hear from others first |
| Blame-shifting | You look weak and unaccountable |
| Panic | You make rushed decisions that make things worse |

## Part 2: The Crisis Communication Framework

When a crisis hits, use this framework: **ACT + FOLLOW**

### ACT (First 24-48 hours)

| Step | Action | Timing |
| :--- | :--- | :--- |
| **A**ssess | Understand what happened, impact, known vs. unknown | First 2-4 hours |
| **C**ontact | Reach out to board/investors directly (phone, not email) | Same day |
| **T**ell | Share what you know, what you don't, and what you're doing | Within 24 hours |

### FOLLOW (Ongoing)

| Step | Action | Timing |
| :--- | :--- | :--- |
| **F**requent updates | Regular communication until resolved | Daily/weekly |
| **O**wn it | Take responsibility, don't deflect | Always |
| **L**earn | Conduct post-mortem, share learnings | After resolution |
| **L**ead | Show calm, decisive leadership | Throughout |
| **O**utcome focus | Keep focus on resolution and path forward | Always |
| **W**atch | Monitor for second-order effects | Ongoing |

## Part 3: Crisis Communication Templates

### Template: The First Call to Your Board Chair

When a significant crisis hits, call your board chair immediately. Here's a script:

```
"[Name], I wanted to call you directly about a situation.

Here's what happened: [Brief, factual description]

Here's what we know so far: [Facts confirmed]

Here's what we don't know yet: [Unknowns]

Here's what we're doing right now:
1. [Immediate action #1]
2. [Immediate action #2]
3. [Immediate action #3]

I'll be updating the full board [timing]. 
Is there anything you'd want me to do differently?"
```

### Template: Crisis Update Email to Board

```
SUBJECT: [URGENT] Update on [Crisis Description]

## Situation Summary
[2-3 sentence overview of what happened]

## Current Status
- What we've confirmed: [facts]
- What we're still investigating: [unknowns]
- Immediate risk level: [High/Medium/Low]

## Actions Taken
1. [Action + owner + status]
2. [Action + owner + status]
3. [Action + owner + status]

## What We Need From the Board
- [Specific ask if any]
- [Decision needed if any]

## Next Update
[Time and date of next communication]

I'm available by phone if you have questions.
```

## Part 4: Specific Crisis Playbooks

### Cash Crisis (Runway < 6 months)

| Action | Timing |
| :--- | :--- |
| Calculate exact runway with scenarios | Day 1 |
| Call board chair | Day 1 |
| Present options to board (cut burn, raise emergency capital, sell) | Within 1 week |
| Develop 90-day survival plan if chosen | Week 2 |

### Key Executive Departure

| Action | Timing |
| :--- | :--- |
| Inform board before public announcement | Immediately |
| Share transition plan and interim coverage | Within 24 hours |
| Present search plan | Within 1 week |
| Provide regular search updates | Weekly until filled |

### Security Breach / Data Loss

| Action | Timing |
| :--- | :--- |
| Inform board and legal counsel | Immediately |
| Assess legal/regulatory notification requirements | First 24 hours |
| Communicate with affected customers as required | Per requirements |
| Share root cause analysis and prevention plan | Within 2 weeks |

## Part 5: Rebuilding Trust After a Crisis

Surviving a crisis is step one. Rebuilding trust takes longer.

| Action | Purpose |
| :--- | :--- |
| Deliver on commitments made during crisis | Prove reliability |
| Share post-mortem and lessons learned | Show learning culture |
| Increase communication frequency temporarily | Demonstrate transparency |
| Acknowledge the impact | Show empathy and accountability |
| Execute your plan | Build track record |

**The trust recovery timeline:** Expect 2-3 quarters of consistent execution before trust fully recovers.
